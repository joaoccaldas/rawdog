# Rawdog Stone Age RPG - Quick Start Guide

## Controls

### Movement
- **WASD** - Move in 4 directions (W=up, A=left, S=down, D=right)
- **Arrow Keys** - Alternative movement

### Tools & Inventory
- **1** - Hand (default)
- **2** - Stone Pickaxe (mine faster)
- **3** - Stone Axe (mine different resources)
- **4** - Spear (combat)
- **5** - Torch (light)
- **6** - HOE (farming)

### Actions
- **C** - Open Crafting menu
- **E** - Eat food / Interact with chests / Harvest crops
- **Left Mouse Click** - Mine resources (hold to mine continuously)
- **Right Mouse Click** - Plant crops (with HOE) / Build blocks
- **Shift + Right Click** - Remove blocks

### Game Features
- **B** - Toggle building mode
- **Tab** - Cycle block types (in building mode)
- **ESC** - Menu / Close panels
- **F5** - Quick save
- **F9** - Quick load
- **Y** - Task assignment (with villagers)
- **T** - Tech tree

## Gameplay

### Mining
1. Equip a pickaxe (press 2 or 3)
2. Hold left mouse click on stone tiles
3. Resources drop when tile is broken

### Farming
1. Equip HOE (press 6)
2. Right-click grass tiles to plant wheat
3. Wait for crops to grow
4. Left-click or press E to harvest

### Survival
- **Health** - Damaged by enemies, starvation, cold
- **Hunger** - Decreases over time, restore with food
- **Temperature** - Affected by environment, proximity to fire

### Crafting
- Press C to open crafting menu
- Craft tools, weapons, and items from resources
- Unlock new recipes as you progress

### Villagers (End-game)
- Build shelters to attract villagers
- Assign tasks with Y key
- Villagers help gather resources

## Tips

- Start by mining stone near spawn
- Build a shelter early to survive cold
- Make a farm for food
- Craft better tools as resources allow
- Explore the world for more resources

## Game Balance Notes

- Mining yields: Stone, Flint, Ochre
- Tool tiers: Hand < Wood < Stone < Advanced
- Food restoration: Cooked Meat (60 hunger) > Raw Meat (15 hunger)
- Tool quality affects mining speed and durability
